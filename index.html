¿Qué es Pop it?

Pop it se traduce literalmente como "revienta esto". Es un juego infantil, cuyo proceso se puede comparar con reventar un envoltorio de burbujas de aire. Pero a diferencia del envoltorio, el pop-it se puede presionar con sensaciones y sonidos característicos "infinitamente": docenas de burbujas de silicona simplemente se empujan hacia el otro lado.

Reglas del juego

Los jugadores se turnan para presionar las burbujas. Es necesario terminar la fila que se empezó. Se pueden presionar máximo 3 burbujas a la vez (deben estar juntas, ¡no se pueden presionar varias burbujas si no están contiguas)! El objetivo es hacer que el oponente presione la última burbuja. ¡Quien presiona la última burbuja, pierde!
